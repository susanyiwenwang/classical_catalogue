About this website

This project is a combination of my past as a classical pianist and my present as a novice programmer.

I've always wondered how I can discover new repertoire, as many piano works are overplayed.

I came upon Open Opus, an API that uses resources from IMSLP. It is not comprehensive, but I have enjoyed trying to use my new skills to do something useful for my musical life.
